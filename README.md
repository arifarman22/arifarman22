## 🚀 Featured Projects

### 🩺 [CureAI — AI Medical Chatbot](https://github.com/arifarman22/CureAI)
> Flask · PostgreSQL · PyTorch · Pinecone · sentence-transformers · HuggingFace API · JWT

A production-grade medical AI chatbot using **RAG architecture** — indexed a 940-page medical textbook into 9,500+ Pinecone vectors for semantic symptom search. Integrates HuggingFace for medical image analysis, scrapes Bangladesh's medex.com.bd for localized medicine recommendations, and is hardened with PBKDF2 hashing (600K iterations), rate limiting, and XSS prevention.

---

### 📊 [AnalytIQ — AI Data Analytics Platform](https://analyt-iq.vercel.app)
> React 18 · FastAPI · PostgreSQL · Pandas · SciPy · Plotly · OpenAI API

Upload a dataset, ask questions in plain English, get back automated **EDA, statistical summaries, and interactive Plotly charts**. Powered by OpenAI API for intelligent recommendations and FastAPI + PostgreSQL backend with JWT auth.
